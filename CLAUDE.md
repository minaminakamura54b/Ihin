# プロジェクト概要
遺品管理AIプラットフォーム。遺族が遺品の写真を撮影・アップロードすると、Claude AIが自動で品物を識別・相場価格を査定する。

# 技術スタック
- Ruby on Rails
- PostgreSQL
- Hotwire（Turbo + Stimulus）
- Active Storage + S3
- Claude API（Anthropic）
- Devise（認証）
- Stripe（決済）
- Renderでデプロイ

# ユーザーのロール
- family（遺族）：無料・メインユーザー
- business（業者）：月額有料
- admin（管理者）：全体管理

# 重要なルール
- コメントは日本語で書く
- Turboを使ってページ遷移を最小限にする
- スマホファーストで設計する
- 感情に寄り添うUIにする（遺族向けのため）
- エラーメッセージは日本語で表示する

# 現在の開発状況
- モデル：完成
- 認証：完成（Devise）
- ルーティング：ほぼ完成
- 業者向けダッシュボード：作成中
- AI査定機能：実装中
- Stripe決済：設定済み・実装中

# やってはいけないこと
- 英語のコメントを書かない
- 不要なgemを追加しない
- DBのマイグレーションを勝手に変更しない

【利用制限の実装】
- 未登録ユーザー：査定は5点まで
- 5点に達したら「続きは無料登録で」
  というモーダルを表示する
- 登録済みユーザー：制限なし
- 制限数はデータベースで管理して
  管理画面から変更できるようにする
